# 📝 To-Do List

Um simples projeto de lista de tarefas para praticar HTML, CSS e JavaScript.

## ✨ Funcionalidades

*   ➕ Adicionar novas tarefas
*   ✔️ Marcar tarefas como concluídas
*   🗑️ Remover tarefas
